## Bibliothèque

Vous pouvez consulter la liste des veeplets disponibles. Un veeplet est une séquence d'instructions permettant de créer un contenu ou un document en une ou plusieurs étapes.

Les veeplets sont classés par catégories et sous-catégories. Par exemple :
* Communication/marketing : vous trouverez les veeplets qui permettent de créer des posts LinkedIn ou des articles de blog
* Transverse : vous trouverez les veeplets qui répondent à plusieurs types de besoins, par exemple le compte-rendu de réunion, la fiche de synthèse...

Si vous souhaitez utiliser un veeplet, il suffit de cliquer sur Go !

## Création d'un veeplet

Si vous souhaitez adapter un veeplet à vos besoins, il faut le personnaliser, ce qui aura pour effet de le copier dans votre espace de travail.

Quand vous êtes sur un veeplet et que vous cliquez sur Personnaliser, un formulaire s'ouvre avec toutes les informations qui le caractérisent. Si vous cliquez sur OK puis sur Enregistrer sans rien modifier, le veeplet sera copié tel quel dans votre espace de travail. Il apparaîtra donc dans la même catégorie, avec exactement les mêmes titres, résumés...

N'hésitez donc pas à modifier le nom, le titre, le résumé, la catégorie du veeplet. Vous pourrez par la suite configurer les instructions fournies à l'IA pour personnaliser la génération des contenus et des documents.

## Mes contenus

Vous pouvez consulter la liste des contenus et documents que vous avez générés. En cliquant sur un contenu, vous accédez aux détails de ce contenu :
* Contenu agrégé : cette vue reprend les éléments générés en un tout.
* Vue comparée : cette vue permet de comparer 2 à 2 ou 3 à 3 les éléments générés.
* Éditer contenu 1..n : ces vues permettent d'éditer et sauvegarder les contenus correspondants. La vue agrégée et la vue comparée sont basées sur ces contenus. Ils pourront par la suite servir de base à d'autres générations.
* Métadonnées : cette vue présente la source, correspondant à la transcription, ainsi que le prompt qui a été utilisé pour transformer le contenu.
