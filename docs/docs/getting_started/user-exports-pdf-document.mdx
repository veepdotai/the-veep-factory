# As a User, Export Your Document as PDF

You can export your artefact as PDF from available templates. They are written in CSS. You can create your own templates if you know this language, otherwise you can ask webmaster to do it for you. 

:::info
(just contact us to be put in touch with the most capable for your situation.) 
:::

## Steps

Options are structured in 3 sections:
* Common Information
* Artefact (Content/Doument) Specific Informations
* Look'n feel (CSS Styles)

## Details

### Common information

Common informations are entered by AI administrator and are used for each pdf export so they are consistent. They are:

* Identification
  - organization name
  - address
  - email
  - website URL
  - id (SIRET in France for example)
* Look'n feel
  - logo
  - header
  - footer
  - background color
  - last cover 
* Default selected elements
  - header: on/off
  - footer: on/off
  - page break before (for h1): on/off
  - table of contents: on/off

### Artefact Specific informations

Specific informations are customized by the user:
* background cover page
* title
* subtitle
* author firstname, lastname, email, phone...
* version number
* date
* background back cover

All elements, except the last one (background back cover) are related to the cover page.
