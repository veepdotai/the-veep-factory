## Veep schema

As you can see in the schema below, Veep is composed of 6 main parts: 

* Specification
* Configuration
* Execution
* Edition
* Publication
* Monitoring

According your role, AI expert or AI user, these parts are interpreted differently. You'll find more information in the [What's Veep part](app/part-overview).
