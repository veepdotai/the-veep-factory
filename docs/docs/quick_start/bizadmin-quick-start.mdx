# Introduction

Le role Prompt Engineer crée des agents. Ces agents sont classés par domaine, sous-domaine, catégorie et sous-catégorie.

Le rôle Biz agent organise le catalogue d'agents IA afin de proposer aux utilisateurs de l'organisation les fonctions IA adéquates en fonction du service de l'entreprise.
