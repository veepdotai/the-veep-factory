# User Journey

Using Veep is very simple once you've used it two or three times.

For now, you just need to go through the documentation by clicking the button below at the bottom of this page to discover the following elements:

* Specification of the business/project or brand context
* Selection of an assistant
* Generation of the expected artifact (content or document)
* Editing the artifact
* Exporting to PDF and/or Publishing
