# Parcours Expert

Il vous suffit de parcourir la documentation en cliquant sur le bouton suivant en bas de cette page pour suivre le Parcours Expert :

* Spécification du contexte de l'entreprise/projet ou de la marque
* Configuration d'un assistant
* Suivi de la génération de l'artefact attendu (contenu ou document)
* Configuration des assistants d'édition
* Configuration des modèles d'exportation (PDF) et/ou cibles de réseaux sociaux
