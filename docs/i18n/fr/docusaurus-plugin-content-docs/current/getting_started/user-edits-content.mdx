# En tant qu'utilisateur, éditez l'artefact généré (Contenu/Document)

Un artefact peut être :
* Contenu : généralement pour des contenus sociaux, qui sont souvent courts. Pensez à eux comme à des publications pour LinkedIn, X, Insta, Facebook...
* Document : généralement pour des rapports, études de cas, livres blancs, articles allant de quelques pages à des ouvrages complets.

## Étapes

* Sélectionnez l'artefact généré
* Modifiez-le, apportez vos mises à jour
* Enregistrez

### Sélectionner un artefact généré

Dans la liste des artefacts, cliquez sur celui que vous souhaitez modifier. Vous pouvez :
* trier le tableau en cliquant sur l'en-tête de la colonne
* rechercher du contenu dans les informations contenues dans le tableau
* sélectionner une catégorie pour restreindre la liste des artefacts (à faire #?)

### Éditer l'artefact

Après avoir cliqué sur un artefact, une fenêtre s'ouvre avec :
* à gauche : la discussion avec l'IA qui fournit les artefacts
* à droite, les éléments qui ont été agrégés pour produire le résultat. C'est votre contenu ou document actuel.

Vous pouvez alors éditer ce résultat. L'objectif est de vous concentrer :
* sur la structure (titre 1...) : c'est pourquoi seules les fonctionnalités gras, italique et souligné sont disponibles. Une fois que vous avez terminé, n'oubliez pas de l'enregistrer. Ne vous attendez pas à trouver des styles à cet endroit.
* sur le texte : relisez le texte, supprimez les redondances, reformulez pour correspondre exactement à vos besoins. Vous pouvez parfois bénéficier d'aide de l'IA et demander à réduire, étendre, simplifier... Vous pouvez également utiliser l'assistant de votre catalogue s'il a été prévu à cet effet.
