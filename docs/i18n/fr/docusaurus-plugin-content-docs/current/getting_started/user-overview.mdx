# Parcours Utilisateur

Utiliser Veep est très simple une fois que vous l'avez utilisé deux ou trois fois.

Pour le moment, il vous suffit de parcourir la documentation en cliquant sur le bouton suivant en bas de cette page pour découvrir les éléments suivants :

* Spécification du contexte de l'entreprise/projet ou de la marque
* Sélection d'un assistant
* Génération de l'artefact attendu (contenu ou document)
* Édition de l'artefact
* Export en PDF et/ou Publication

:::tip
Vous souhaitez entrer en contact avec des experts en IA ? Il vous suffit de <a href="contact\@veep.ai">nous contacter</a>. Nous vous répondrons rapidement !
:::
