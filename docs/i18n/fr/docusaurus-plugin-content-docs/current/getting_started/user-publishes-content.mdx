# En tant qu'utilisateur, Publier du contenu

La publication peut prendre diverses formes selon vos objectifs :
* Document PDF à envoyer en pièce jointe
* Document à archiver dans votre ECM préféré (Google Drive, Sharepoint ou Office 365, NextCloud...) ou lien dans votre application préférée (ERP, CRM...)
* Page web à publier sur votre site web
* Publication sur LinkedIn, avec carrousel, à publier sur votre page personnelle ou d'entreprise
* Story Insta, à publier sur votre compte Insta
* Publication X
* Article à publier sur LinkedIn, Medium, votre site web...

## Étapes

Pour chacune de ces publications, vous devez d'abord définir des cibles. Une cible est définie à partir des informations suivantes :
* URL du post
* nom d'utilisateur
* mot de passe ou jeton
