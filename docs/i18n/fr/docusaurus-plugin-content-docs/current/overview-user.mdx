# Veep.AI vous permet de construire, déployer et monétiser votre IA verticale.

:::info
Veep propose une solution qui permet un déploiement progressif de l'IA dans votre entreprise, service par service. Elle propose un catalogue de fonctionnalités IA prêtes à l'emploi, pour gagner en productivité et créer de la valeur au niveau de l'entreprise et pas seulement de l'individu.

Elle est open source, vous pouvez donc utiliser votre propre architecture si vous le préférez.

:::
