En tant qu'utilisateur, exportez votre document en PDF

Vous pouvez exporter votre artefact au format PDF à partir des modèles disponibles. Ils sont écrits en CSS. Vous pouvez créer vos propres modèles si vous maîtrisez ce langage, sinon vous pouvez demander au webmaster de le faire pour vous.

info

(contacter-nous simplement pour être mis en relation avec la personne la plus compétente pour votre situation.)

Étapes​

Les options sont structurées en 3 sections :

  • Informations communes
  • Informations spécifiques à l'artefact (contenu/document)
  • Apparence (Styles CSS)

Détails​

Informations communes​

Les informations communes sont saisies par l'administrateur de l'IA et sont utilisées pour chaque exportation PDF afin qu'elles soient cohérentes. Elles sont :

  • Identification
    • nom de l'organisation
    • adresse
    • e-mail
    • URL du site web
    • id (SIRET en France par exemple)
  • Apparence
    • logo
    • en-tête
    • pied de page
    • couleur d'arrière-plan
    • dernière couverture
  • Éléments par défaut sélectionnés
    • en-tête : on/off
    • pied de page : on/off
    • saut de page avant (pour h1) : on/off
    • table des matières : on/off

Informations spécifiques à l'artefact​

Les informations spécifiques sont personnalisées par l'utilisateur :

  • page de couverture arrière
  • titre
  • sous-titre
  • prénom, nom, e-mail, téléphone de l'auteur...
  • numéro de version
  • date
  • couverture de l'arrière-plan

Tous les éléments, sauf le dernier (couverture arrière), sont liés à la page de couverture.
